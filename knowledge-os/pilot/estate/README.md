# SME pilot — estate agency (Knowledge OS + Skillforge)

**Goal:** Validate a *small-business* bundle: local Obsidian + meta registry + **human-approved** inbox routing + daily index refresh, with vertical content via packaged skills and memory. Not a full DMS — files stay on disk; collaboration is "git or Obsidian Sync + policy," not real-time co-editing.

## What we ship in this folder

| Artifact | Role |
|----------|------|
| [vault-skeleton.md](vault-skeleton.md) | Suggested tree for `meta` + one line-of-business vault (`listings/`) |
| [cowork-schedule.md](cowork-schedule.md) | Minimum Cowork cadence and success criteria |

## Pilot acceptance criteria

- Time from raw note in `meta/inbox` to file in the correct domain vault (median, self-reported).
- Super-wiki row freshness after a domain change (`_os/log.md` drives refresh).
- One trained user completes Cowork "Inbox process" and "Super wiki refresh" without hand-editing generated indices.
