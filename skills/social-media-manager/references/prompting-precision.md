# Prompt precision (LLM instructions)

## Goals

- Turn vague requests into clear, actionable instructions.
- Reduce ambiguity before drafting content or code.
- Align prompts with plan → act → validate workflows.

## Workflow

1. **Analyze** intent; flag missing context, constraints, or output shape.
2. **Clarify** with targeted questions (audience, scope, definition of done).
3. **Draft** a structured prompt: goal, inputs, steps, success criteria.
4. **Optimize** for token efficiency — cite paths and symbols explicitly.

## Guidelines

- Replace vague verbs ("fix", "clean up", "improve") with measurable outcomes.
- Include file paths, interfaces, and business rules when relevant.
- Specify output format, tone, and verification steps.
- Prefer deterministic checklists over open-ended asks.
