# Platform Formats Reference

Read when drafting platform-specific content to apply correct character limits,
formatting conventions, and best practices.

---

## X / Twitter

| Attribute | Value |
|---|---|
| Character limit | 280 per post; threads have no total limit |
| Image | Up to 4 images; 1200×675px recommended |
| Video | Up to 2 min 20 sec; MP4 preferred |
| Hashtags | 1–2 maximum; more reduces engagement |

**Format tips:**
- Hook in the first line — no context before the value
- Use line breaks sparingly; compress into dense sentences
- Threads: end each post with 🧵 N/N or similar thread indicator
- Alt text: always add for images

---

## LinkedIn

| Attribute | Value |
|---|---|
| Character limit | 3000 (posts); 700 (comments) |
| Image | 1200×627px for link previews; 1080×1080px for standalone |
| Video | Up to 10 min; square (1:1) or vertical (9:16) performs well |
| Hashtags | 3–5 at end of post |

**Format tips:**
- Line breaks every 1–2 sentences — dense paragraphs are skipped
- First 3 lines are visible before "see more" — hook must land there
- Personal narrative and lessons outperform product announcements
- Tag people and companies sparingly — only when genuinely relevant

---

## Instagram

| Attribute | Value |
|---|---|
| Caption limit | 2200 characters |
| Image | 1080×1080px (square), 1080×1350px (portrait) |
| Reels | Up to 90 seconds for standard; up to 15 min via app |
| Hashtags | Up to 30; 5–10 highly relevant hashtags recommended |

**Format tips:**
- Visual comes first — the caption supports the image, not the other way around
- First sentence is the hook; details in the body
- Call to action at the end (comment, save, visit link in bio)
- Stories: interactive polls and questions drive engagement

---

## Threads (Meta)

| Attribute | Value |
|---|---|
| Character limit | 500 per post |
| Image | Multiple images per post supported |
| Video | Up to 5 minutes |
| Hashtags | Limited support; conversational tone preferred |

**Format tips:**
- Very similar to X in tone — short, direct, conversational
- Threads (reply chains) work well for longer ideas
- Cross-posting from X/Twitter works but adjust for the audience

---

## Bluesky

| Attribute | Value |
|---|---|
| Character limit | 300 per post |
| Image | Up to 4 images with alt text |
| Video | Limited — embed via links |
| Hashtags | Supported; use 1–2 per post |

**Format tips:**
- Audience skews open-source, developer, and tech-adjacent
- Authentic, non-corporate tone performs well
- Link previews (cards) are prominently displayed

---

## YouTube

| Attribute | Value |
|---|---|
| Title | 70 chars max (60 recommended for SEO) |
| Description | 5000 chars; first 150 shown in search |
| Tags | 500 chars total; use 5–8 specific tags |
| Thumbnail | 1280×720px; JPG/PNG; text overlay improves CTR |
| Chapters | Add timestamps in description for videos >5 min |

**Format tips:**
- Title: searchable keyword + value statement (e.g. "How to X in 5 Minutes")
- Description: first 2–3 sentences summarise the video; include keywords naturally
- Chapters improve watch time and search ranking for longer videos
- End screen: add subscribe CTA and suggest 2 related videos
